# Text File Statistics Program (.NET)

This application, written in .NET, analyzes a text file and provides various statistics about its content.

### Features 
-  Unique Word Count: The number of distinct words found in the file.
- Word Count: The total number of words found in the file (including duplicates).
- Character Count: The total number of characters (including spaces and punctuation) in the file.
- Line Count: The total number of lines in the file.

### How to run
1. Clone the Github Repo
2. Make sure .NET and the .NET CLI are installed (If not go download .NET from https://dotnet.microsoft.com/en-us/download)
3. Run `dotnet run` in your terminal to start application
4. In the application enter the text files path (Ex.  `/User/Documents/hello.txt`)
